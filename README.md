# Portfolio Website

This is a simple portfolio website built with HTML, CSS, and JavaScript.

## Deployment on GitHub Pages

1.  **Push to GitHub:** Push the code to a new repository on your GitHub account.
2.  **Enable GitHub Pages:** In your repository's settings, go to the "Pages" section.
3.  **Select Branch:** Under "Source", select the `main` branch (or whichever branch you pushed your code to).
4.  **Save:** Click "Save". Your website should be live in a few minutes at `https://<your-username>.github.io/<your-repository-name>/`.
